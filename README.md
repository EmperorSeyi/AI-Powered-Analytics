# AI-Powered-Analytics
- Combined OpenAI API + LLM + Python + SQL+ML to build an hybrid AI-powered analytic and decision-making system that transforms natural language queries into SQL codes that generate  tabular datasets to build predictive ML models for manufacturing problems
- Evaluated the model using confusion matrix, accuracy, precision, recall, f1-score and ROC-AUC
- Developed AI-interpretation layer that transforms model output into operational insights and business recommendations.
- Developed a prompt log, technical report and presentation slide
